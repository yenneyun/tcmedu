# 圣手学堂 · 经络推拿师内训系统 (专家增强版) - 产品规格说明书

## 1. 项目愿景
打造一个“理论-实操-考核”闭环的数字化内训平台，通过规范化的教学和量化评价，培养具备名家思维的顶级推拿师。

## 2. 核心架构
- **前端**: React 18, Tailwind CSS, Lucide React (图标), Motion (动画)
- **数据存储**: LocalStorage (单机、多用户本地数据持久化)
- **适配**: 完美适配平板与桌面端，兼容移动端查询。

## 3. 手法逻辑库 (核心定义)
### 3.1 放松类手法
- **一指禅推法**: 沉肩垂肘，腕部摆动。作用：疏经通络，调理脏腑。
- **滚法**: 手背吸附。作用：大面积接触，解痉止痛。
- **揉法**: 环旋动。作用：消肿化瘀。
- **拿法**: 捏放结合。作用：舒筋解表。
- **拨法**: 横向拨动肌腱。作用：松解粘连。
- **搓法**: 对掌用力搓。作用：理顺肌纤维，温通气血。
- **牵拉法**: 缓慢拉伸。作用：改善关节角度。
- **击法**: 虚拳或掌击。作用：行气活血。
- **梳头法**: 十指如梳。作用：清脑安神。

### 3.2 温通类手法
- **擦法**: 直线往返，产热为度。作用：温补肾阳。
- **推法**: 沿经络单向推进。作用：疏通经气。
- **抹法**: 柔和贴服抹。作用：宽胸理气。
- **拍法**: 虚掌拍击。作用：宣肺排痰。
- **鸣天鼓**: 弹击枕部。作用：耳聪目明。
- **刮法**: 边缘刮拭。作用：解表散寒。

## 4. 治疗原则定义
- **强则松之**: 肌肉痉挛、张力过高。手法：滚、揉、牵拉。
- **瘀则祛之**: 局部青紫、刺痛、血行不畅。手法：拨、拍、抹。
- **塞则通之**: 经络闭阻、肢体麻木。手法：一指禅、点、拨。
- **肿则消之**: 局部水肿。手法：轻摩、轻擦。
- **寒则温之**: 畏寒喜暖。手法：重擦、摩擦。
- **失则调之**: 脏腑功能紊乱。手法：按、摩、震。
- **凝则动之**: 关节僵硬、活动受限。手法：关节动法。
- **聚则展之**: 腹部痞块、肌肉硬结。手法：深揉、拨。
- **乱则复之**: 关节错缝。手法：正骨复位。
- **收则散之**: **治疗结束时，旨在宣散气血，防止气聚局部。** 手法：搓、指尖击。

## 5. 新增重点方案：梨状肌综合征
- **定义**: 梨状肌受损压迫坐骨神经。
- **治法**: 舒筋活络、松解止痛。
- **穴位**: 环跳、秩边、阿是穴。
- **核心操作**: 在梨状肌处使用拨法，伴随髋关节外旋。
